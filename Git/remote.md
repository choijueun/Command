# Remote

> git remote

git remote 저장소 확인
- -v: URL 표시
<br/>

> git remote show (remote명)

git remote 저장소의 상세정보 확인   
<br/>

> git remote add (remote명) (URL)

git remote 저장소 추가   
<br/>

> git remote remove (remote명)

git remote 저장소 삭제   
<br/>

> git remote rename (old) (new)

git remote 저장소 이름 변경   
<br/>
